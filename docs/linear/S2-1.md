# S2-1 — TeacherBroker + redaction strict (no raw chunks)

## Scope
- Added `teacher_broker.py` with strict request building for teacher calls:
  - question + metadata + redacted summaries only
  - raw chunk marker blocking (`BEGIN_CONTEXT*` markers)
- Added timeout-safe teacher call wrapper (`asyncio.wait_for`) with fallback payload.
- Integrated TeacherBroker in missing-context paths (chat/RAG/graph) to enrich guidance.
- Added telemetry counters:
  - `teacher_calls`
  - `teacher_blocked_redaction`

## Implementation notes
- Teacher requests never include raw chunk fences.
- If redaction/marker policy blocks content, broker reports blocked status and metric increments.
- On timeout/error, code falls back to safe `{"timeout_or_error": true}` guidance.

## Validation
- Added unit tests for broker marker blocking and safe request shape.
- Added static wiring tests for metrics and broker integration points.
