# S1-4 — Citations contractuelles `path@commit` + validator anti-bidon

## Scope
- Enforced citation contract metadata with explicit `citation_contract=path@commit` in retrieval stats.
- Added support-check validator (`answer_supported_by_context`) to detect unsupported generated answers against retrieved passages.
- If support check fails, response is downgraded to standardized `missing_context` payload.
- Added telemetry for unsupported/fake-citation downgrade via `false_citation_rate` (`record_false_citation_event`).

## Implementation notes
- Validation is applied on:
  - chat-completions RAG path
  - `/v1/rag/query`
  - `/v1/rag/query/graph-enriched`
- Downgrade keeps deterministic citation contract and flags:
  - `response_mode=missing_context`
  - `support_check_failed=true`
