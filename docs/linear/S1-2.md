# S1-2 — Reranker obligatoire (cheap) + config

## Scope
- Added a cheap reranker module (`cheap_reranker.py`) that computes `rerank_score` and blends it with vector score into final ranking score.
- Integrated reranking as an explicit stage after hybrid retrieval in:
  - `search_documents`
  - `search_with_graph_enrichment` vector retrieval phase
- Added configuration knobs:
  - `RERANK_ENABLED` (default true)
  - `RERANK_ALPHA` (default 0.65)
  - `RERANK_MAX_CANDIDATES` (default 40)
  - `RERANK_ALPHA_BY_WORKSPACE` (JSON map override)

## Implementation notes
- Workspace-specific rerank tuning is supported via `RERANK_ALPHA_BY_WORKSPACE`.
- Each reranked result carries `rerank_score` and `ranking_stage="reranked"`.
- Final ranking score is blended score after rerank stage.

## Validation
- Added unit tests for reranker behavior.
- Added static wiring tests for reranker config and integration.
