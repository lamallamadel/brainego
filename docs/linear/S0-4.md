# S0-4 — Golden set v1 (50 questions) + eval runner

## Scope
- Added `tests/contract/fixtures/golden_set_v1.json` with 50 curated baseline questions.
- Added `scripts/golden_set_v1_runner.py` to compute and emit:
  - `grounded_accuracy_answerable`
  - `false_citation_rate`
  - `missing_context_actionable_rate`
- Runner writes a JSON report and prints a console report.

## Input/output contract
- Input golden set: JSON object with `version` and `cases`.
- Input outputs: JSON list (or `{ "outputs": [...] }`) keyed by case `id`.
- Output report: JSON with `metrics` + supporting `counts`.

## Validation
- Added unit tests for metric computation and fixture shape/size (50 cases).

## Example command
```bash
python scripts/golden_set_v1_runner.py \
  --golden-set tests/contract/fixtures/golden_set_v1.json \
  --outputs tests/contract/fixtures/golden_set_v1_outputs_sample.json \
  --report artifacts/golden_set_v1_report.json
```
