# S0-1 — IntentClassifier v1 (must_ground / should_ground / freeform)

## Scope
- Added a dedicated grounding intent classifier with three policies:
  - `must_ground`
  - `should_ground`
  - `freeform`
- Wired classification into both `/v1/chat` and `/v1/rag/query`.
- Added telemetry for `intent_distribution` in API metrics.

## Implementation notes
- Heuristic classifier implemented in `grounding_intent_classifier.py` with explicit coverage for:
  - greetings
  - factual internal/doc questions
  - document summary requests
  - brainstorming prompts
- `/v1/chat` now appends `grounding_intent` inside `x-routing-metadata`.
- `/v1/rag/query` now computes `grounding_intent` from `request.query` and surfaces it in `retrieval_stats`.
- `MetricsStore` now tracks `intent_distribution` counters and exposes them through `/metrics/json` payload.

## Validation
- Unit tests added for classifier behavior.
- Static endpoint wiring tests updated for grounding metadata and telemetry hooks.
