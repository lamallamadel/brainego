# S0-3 — MissingContextResponse standard + questions ciblées

## Scope
- Added a standardized `missing_context` response payload with:
  - `type=missing_context`
  - `reason=insufficient_evidence`
  - `targeted_questions` (max 3)
  - `ingestion_suggestion` (single actionable recommendation)
- Added gating policy:
  - if `grounding_intent == must_ground` and `ESS < ESS_THRESHOLD_HIGH`
  - return missing-context response without LLM generation call.
- Added telemetry counter for `missing_context_rate`.
