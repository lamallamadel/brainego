# S0-2 — Evidence Sufficiency Score (ESS) v1

## Scope
- Added `compute_evidence_sufficiency` to compute ESS in `[0,1]` from retrieval scores and source diversity.
- Added configurable thresholds:
  - `ESS_THRESHOLD_LOW` (default `0.35`)
  - `ESS_THRESHOLD_HIGH` (default `0.60`)
- Exposed ESS in retrieval debug stats for RAG endpoints.
- Added `ess_histogram` telemetry in metrics store.

## Implementation notes
- ESS is computed for:
  - chat-completions RAG branch metadata (`rag_metadata`)
  - `/v1/rag/query` retrieval stats
  - `/v1/rag/query/graph-enriched` retrieval stats
- `MetricsStore` now keeps ESS samples and a bucketized histogram for observability.

## Validation
- Added static unit tests validating ESS module, threshold wiring, and API ESS integration.
